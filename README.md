# 草野球ログ MVP

自分専用の草野球記録アプリです。ブラウザだけで動き、試合登録、打席入力、主要な打撃成績の自動集計ができます。

## できること

- 試合種別、対戦相手クラス、球場、試合結果、打順、守備位置、守備・走塁メモを保存
- ホームで最近の試合を選び、守備・走塁・試合メモを1試合分ずつ確認
- 打席、相手投手、背番号、投球フォーム、ランナー、ストレート球速、変化球、盗塁、得点、カウント、打席結果と勝負球情報を保存
- スマホ入力タブで、試合後の打席を大きめのボタン中心で素早く保存
- サインを `なし / バント / エンドラン / スクイズ` から選択
- 試合・打席入力で登録済みの試合を選び、その試合の打席だけを確認
- 打率、本塁打、打点、得点、盗塁数、盗塁成功率、出塁率、長打率、得点圏打率、OPS、犠飛、犠打を集計
- 打球方向の入力データから打球方向比率を図で表示
- 対投手別、球種別、コース別、カウント別に集計
- コース別成績・補足で9分割のコース別打率を色分け表示
- 相手投手データは投手名、対戦相手、背番号、左右、フォーム、球速、変化球のボタン一覧から過去打席と対戦成績を確認
- データの書き出し、読み込み
- 保存成功・失敗のステータス表示
- 保存時の自動バックアップ作成、最新バックアップの書き出し・復元
- 登録済みの試合・打席をフォームへ読み込んで編集保存
- Supabaseにログインして、MacとiPhoneのデータを自動同期

## 使い方

`index.html` をブラウザで開くと使えます。データはブラウザ内に保存されます。
保存に成功すると画面上部に「保存済み」と表示され、自動バックアップも作成されます。

## Supabase同期の準備

1. Supabaseでプロジェクトを作成します。
2. アプリの「同期設定」タブに表示されているSQLを、SupabaseのSQL Editorで実行します。
3. SupabaseのProject URLと anon / publishable key をアプリの「同期設定」に入力します。
4. メールアドレスとパスワードで新規登録またはログインします。
5. ログイン後は、保存時にクラウドへ自動同期されます。

パスワードはアプリには保存しません。Supabase未設定でも、今まで通りブラウザ内保存だけで使えます。

## iPhoneで毎回開きやすくする

このアプリはホーム画面追加用の設定を入れています。
GitHub PagesなどのHTTPS公開URLで開き、Safariの共有ボタンから「ホーム画面に追加」すると、アプリのように起動できます。

詳しい公開手順は [PUBLISH.md](./PUBLISH.md) を見てください。
